# RSAW Bound Estimation under Uncertain Weights

This repository contains the GAMS implementation of the Relative Simple Additive Weighting (RSAW) estimation framework under interval uncertainty in criteria weights.
The code provides lower and upper bounds for RSAW scores, enabling robust ranking of alternatives in multi-criteria decision-making (MCDM) when the exact importance of criteria is not known.

The RSAW method extends SAW by incorporating relative scaling between alternatives, and this repository provides:
